This repository was created using the R package of 'manuscript' which aims to facilitate the process of writing scientific manuscripts.

## Organization of the this repository

- data/: folder that store the data that created the manuscript.
  - data/raw: to store the raw data
  - data/clean: to store the clean data
- rscripts: store the R scripts and functions, used for data cleaning or rmd or qmd manuscripts.
- output: store the output of the manuscript.
